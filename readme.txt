API para chatbot con IA usando express con node 

La arquitectura del proyecto es MVC, no se usa ningún framework adicional

Antes de ejecutar el código, tener:

-La base de datos SQL local llamada dev_dsd
-La base de datos SQLite para manejar las sesiones de los usuarios 
-El archivo .Env con los claves 


Comando inciales

-npm i
-npm node main.js
